# 📱 Trackly – Personal Finance Tracker

Trackly is a student-friendly personal finance tracking mobile application built using **React Native**, **Expo**, and **TypeScript**. It helps users manage expenses, monitor spending habits, and set budgets through a clean and intuitive interface.

## 🚀 Features

- Add and manage daily expenses
- Category-wise spending analysis
- Budget setting and tracking
- Spending history
- User authentication
- Personal profile management
- Mobile-first responsive design
- Clean and modern UI

## 📸 Screenshots

### Overview Dashboard
Shows total spending, recent expenses, and category-wise spending insights.

### Budget Management
Set monthly budgets for different spending categories.

### User Profile
View account details and financial statistics.

## 🎯 Project Goal

The goal of Trackly is to help users become more aware of their spending habits and make smarter financial decisions through simple expense tracking and budgeting tools.
